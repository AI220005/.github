# SAMK-Applied-Programming

**SAMK-Applied-Programming** is the official GitHub organization for the *Applied Programming* course at [Satakunta University of Applied Sciences (SAMK)](https://www.samk.fi).

This organization contains course repositories, templates, and example code used in the Applied Programming course within the Data Engineering and Artificial Intelligence degree programmes.

Students enrolled in the course will use these repositories as part of their coursework.
